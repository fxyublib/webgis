

1. 开启服务器: 鼠标双击 nginx-ev.exe 即可启动;
   (如果防火墙授权弹出请求提示,请点击"允许");

2. 关闭服务器: 鼠标双击 nginx-ev-stop.bat 即可退出;


-----------------------------------------------------

LiveView 直播:

1. 在设置界面的直播地址中填上:
   rtmp://192.168.1.100/live/my       (IP 地址可以在cmd中通过命令行ipconfig查询)


观看直播:
2. 在浏览器中输入地址:  localhost:8080   按回车,页面会跳转到播放器;
   在播放器地址栏中填写:  rtmp://192.168.1.100/live/my   点击 Play...按钮即可播放;
